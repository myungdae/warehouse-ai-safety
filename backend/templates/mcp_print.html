🤖 MCP 자동화 흐름 — Operational Ontology + Agentic AI
🚁
1. 야간 드론 순찰
설정된 시각에 드론 A·B 자동 출동. Rack-by-Rack 스캔. 유선 전원으로 무제한 운용.
🤖
2. AI 분석 & Rescan
ERP Day1 vs Day2 비교, 분실·감소 감지, 자동 Rescan 판정 및 우선순위 분류.
💾
3. 보고서 자동 저장
AI 권고사항·재스캔 결과·변동 내역 포함한 종합보고서 DB 저장. Report Admin 즉시 등록.
🖨️
4. 자동 출력 (MCP)
프린터 전원 확인 후 지정 시각에 보고서 자동 출력. 담당자 출근 시 이미 인쇄 완료 상태.
⚠️ 프린터 전원 조건: 자동 출력이 정상 작동하려면 프린터가 네트워크에 연결된 상태(전원 ON + LAN 연결)여야 합니다. 프린터가 오프라인이면 출력을 건너뛰고 로그에 기록합니다. 이후 수동 출력 버튼으로 재시도 가능합니다.
