🔍 IFF — Identification Friend or Foe (아군·적 식별)

탐지된 드론이 누구의 것인지 먼저 판별합니다. 이 판별이 틀리면 아군이나 민간인에게 교전 명령이 내려질 수 있으므로, 가장 먼저, 가장 신중하게 처리합니다.

✅ FRIENDLY — 아군

아군 드론 · 교전 절대 금지

우리 측에서 운용 중인 드론. IFF 트랜스폰더 코드로 자동 확인됩니다.

📡 IFF 트랜스폰더 코드 응답 확인
🔑 사전 등록된 아군 코드 DB 매칭
⛔ ROE 무관하게 교전 명령 SKIP
🏙️ CIVILIAN — 민간

민간 상업 드론 · 교전 절대 금지

DJI 등 일반 드론. 국가 드론 등록 시스템 번호 또는 주파수로 확인.

🪪 민간 등록번호 (KOR-, CIV-, DJI-) 패턴
📻 ISM 주파수 (2.4GHz / 5.8GHz) + 저속(15m/s 미만)
⛔ ROE 무관하게 교전 명령 SKIP
🔴 HOSTILE — 적

적 드론 · 교전 가능 (ROE에 따름)

적대 세력의 드론. 여러 특성이 복합적으로 적 드론을 가리킬 때 판정.

📡 IFF 무응답 (아군 코드 없음)
🚀 고속 (20m/s 초과)
🔥 고열 신호 (열화상 0.5 초과)
⚡ 조건 3가지 동시 충족 시 HOSTILE 판정
❓ UNKNOWN — 미상

식별 불가 · ROE에 따라 처리

위 세 가지에 해당하지 않는 드론. IFF 무응답이지만 HOSTILE 기준 미달.

📡 IFF 코드 없음 · 등록번호 없음
⚠️ HOSTILE 기준 미충족 (저속 또는 저열)
🔎 위협도 점수 기반으로 ROE가 최종 결정
⚡ IFF 판정 순서 (폭포수 방식)
1️⃣
IFF 코드 확인
아군 코드 DB에 있으면 → 즉시 FRIENDLY 확정, 이후 단계 건너뜀
2️⃣
등록번호 확인
민간 등록번호 패턴이면 → 즉시 CIVILIAN 확정
3️⃣
주파수+속도 확인
민간 ISM 주파수(2.4/5.8GHz) + 저속(15m/s 미만) → CIVILIAN
4️⃣
복합 HOSTILE 판정
IFF 무응답 AND 고속(20m/s↑) AND 고열(0.5↑) → HOSTILE
5️⃣
나머지는 UNKNOWN
위 1~4 모두 해당 없음 → UNKNOWN으로 분류, ROE+위협도로 최종 결정
📋 ROE — Rules of Engagement (교전규칙)

IFF로 식별이 된 후, "지금 상황에서 교전해도 되는가?"를 결정하는 규칙입니다. 전장 상황에 따라 지휘관이 실시간으로 변경할 수 있습니다.

IFF 결과 ↓ / ROE → 🔓 WEAPONS_FREE 🔒 WEAPONS_TIGHT 🚫 WEAPONS_HOLD
✅ FRIENDLY (아군) SKIP ⛔ SKIP ⛔ SKIP ⛔
🏙️ CIVILIAN (민간) SKIP ⛔ SKIP ⛔ SKIP ⛔
🔴 HOSTILE (고위협) ENGAGE 🔴 ENGAGE 🔴 HOLD ⏸️
🔴 HOSTILE (중위협) INTERCEPT 🟠 INTERCEPT 🟠 HOLD ⏸️
❓ UNKNOWN (고위협) ENGAGE 🔴 MONITOR 🟡 HOLD ⏸️
❓ UNKNOWN (저위협) MONITOR 🟡 MONITOR 🟡 HOLD ⏸️

SKIP: 교전 면제 (아군/민간)
